Instrucciones de Github:

git init -> inicializa un repo vacío 
git add .
git commit -m "first commit"
git branch -M main --> si aún no se ha realizado ningún commit la rama main no existe
git remote set-url https://github.com/Usuario_github/nombre_repositorio.git
git push -u origin main


Crear una nueva rama:
git branch nombre_rama
git checkout nombre_rama --> moverte a esa rama

git checkout -b nombre_rama --> este es un atajo que hace las dos instrucciones anteriores

git branch --> muestra las ramas de un repositorio
